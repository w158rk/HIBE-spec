## 默认参数

每一个版本的系统标准都会指定一组默认的协议规则和参数，下表为该版本建议的默认参数，实现者****可以****选择与下表不同的参数。

|   参数   |    描述    |   建议的默认值    |
|----------|------------|-----------------|
| `HASH_ALGORITHM` {#hash-algorithm} | 用于计算哈希值的算法 | SM3 |
| `DATA_ENCODING_SCHEME` { #data-encoding-scheme } | 用于数据编码，编码结果 ****必须**** 是ASCII形式 | Base64 |
